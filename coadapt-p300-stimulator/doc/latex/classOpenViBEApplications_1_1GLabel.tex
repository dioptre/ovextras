\subsection{Detailed Description}
Base class of \hyperlink{classOpenViBEApplications_1_1GPictureSymbol}{GPictureSymbol} and \hyperlink{classOpenViBEApplications_1_1GSymbol}{GSymbol} that handles the common properties such as foreground color, filename of the source file (font file for \hyperlink{classOpenViBEApplications_1_1GSymbol}{GSymbol}, png file for \hyperlink{classOpenViBEApplications_1_1GPictureSymbol}{GPictureSymbol}) label size and label position TODO maybe some of these constructors should be protected as we should always construct an object of the derived classes
