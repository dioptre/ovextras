Detailed Description

A label class that does not have letters as content such as GSymbol, but pictures coming from png files

Constructor & Destructor Documentation

GPictureSymbol::GPictureSymbol ( OpenViBE::CString  sourcePath,
OpenViBE::float32  scaleSize 
)

has to create a texture manager to make sure all buttons that use the same picture use the same resource in graphical memory so that no multiple textures of the same pic are created
On top of that it also uses display lists managed by OpenGLDListManager, both are created by the super class GLabel. One display list is filled in by GLabel, the other one by this class

Parameters:
sourcePaththe file name of the png file that will be used to make the texture
scaleSizea number between 0 and 1 that defines the size of the texture in the button box
