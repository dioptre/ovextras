Detailed Description

This class uses the ISharedMemoryReader 'library'/interface to read predictions and letter probabilities as computed by the P300 scenario from the shared memory

void ExternalP300SharedMemoryReader::openSharedMemory ( OpenViBE::CString  sharedMemoryName)

Simply constructs a SharedVariableHandler to provide access to all the variables in shared memory
It is assumed that the first variable will be the stimulus (corresponding to the letter that is predicted by the P300 scenario) and the second variable could be the matrix with letter probabilities.
So the shared memory writer box in the scenario should have at least a stimulation input and optionally a secondary streamed matrix input.
